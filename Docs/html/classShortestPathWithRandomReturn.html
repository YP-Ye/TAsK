Detailed Description

Implements randomised flow update strategy for path-based algorithms.

Some of the point-to-point shortest path calculations are skipped. The probability of performing a point-to-point shortest path calculation depends on the current iteration: it is equal to one for the first iteration and to . This class works with any kind of point-to-point shortest path: A*, non-additive shortest path, etc.
