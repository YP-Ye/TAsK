Detailed Description

This class implements a bush - basic element of origin-based algorithms.

This class implements a factory method - children classes allocate necessary DAGraph objects. DAGgraph is not used directly but through OriginBush because some operations are common (so it reduces code duplication) also some of the operations can be over-ridden by children classes. For example, see OriginBushTapas.
