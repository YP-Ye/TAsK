void PieceWiseLinearIndiffCurve::addPoint ( TollType  toll,
FPType  value 
)

This method is used for scalarization function creation. Adds point (toll, value) to the scalarization function.

Note
Points must be added in the increasing order of toll. Points with the same value of toll cannot be added. All values must be non-negative. The first value of toll must be zero. If these conditions are violated, the error will be thrown.
FPType PieceWiseLinearIndiffCurve::evaluate ( TollType  toll)
virtual

Evaluates the scalarization function given the value of toll. If toll is between two known points, the returned value corresponds to a line between these two points. If toll is more than last known point, maximum value of FPType is returned. If toll is negative, assert will terminate the program.
